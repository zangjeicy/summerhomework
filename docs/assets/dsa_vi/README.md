# 金股睿析品牌资产

本目录保留“金股睿析 / RuyiDailyStockAnalysis”的可编辑源资产。新标志将“如意云纹”与“向上走势”合并为一个简洁符号：白色曲线代表如意与数据云，蓝绿走势代表量化分析与可验证的增长。

## 文件

- `icon.svg`：方形主图标，适用于应用图标、侧边栏和 favicon。
- `logo-light.svg`：浅色背景横版 Logo。
- `logo-dark.svg`：深色背景横版 Logo。
- `icon-16.png`、`icon-32.png`、`icon-180.png`、`icon-512.png`：常用尺寸导出。
- `favicon.ico`：16/32/48/64/128/256 多尺寸浏览器图标。

Web 运行时副本位于 `apps/dsa-web/public/brand/`，入口 favicon 位于 `apps/dsa-web/public/favicon.svg`。

## 使用规范

- 主色：深海蓝 `#0B1F45`、量化蓝 `#174BA5`、数据青 `#087F9E`、增长绿 `#70F0B5`。
- 图标周围至少保留图标宽度 `1/8` 的留白，不拉伸、不改变配色、不叠加阴影或描边。
- 小于 24px 时只使用 `icon.svg`，不要使用横版 Logo。
- 浅色背景使用 `logo-light.svg`，深色背景使用 `logo-dark.svg`。
