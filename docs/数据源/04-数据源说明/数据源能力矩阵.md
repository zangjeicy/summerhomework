# 数据源能力矩阵

本矩阵依据当前 `DataFetcherManager._DAILY_MARKET_FETCHER_SUPPORT`、默认初始化代码和各适配器入口整理。它描述已经接入代码的能力，不代表第三方供应商全部能力。

| 数据源 | 日线市场 | 实时行情 | 基本面 | 新闻 | 密钥要求 | 主要代码 |
| --- | --- | --- | --- | --- | --- | --- |
| EFinance | A股 | A股 | 否 | 否 | 否 | `data_provider/efinance_fetcher.py` |
| Tencent | A股 | A股 | 否 | 否 | 否 | `data_provider/tencent_fetcher.py` |
| AkShare | A股、港股 | A股、港股 | A股适配器 | 否 | 否 | `data_provider/akshare_fetcher.py` |
| Tushare | A股、港股 | A股 | 否 | 否 | 是 | `data_provider/tushare_fetcher.py` |
| TickFlow | A股 | A股 | 否 | 否 | 是 | `data_provider/tickflow_fetcher.py` |
| Pytdx | A股 | 未作为统一实时主路由 | 否 | 否 | 否 | `data_provider/pytdx_fetcher.py` |
| Baostock | A股 | 否 | 否 | 否 | 否 | `data_provider/baostock_fetcher.py` |
| YFinance | A股、港股、美股、日股、韩股、台股 | 港美股等 | 港美股适配器 | 否 | 否 | `data_provider/yfinance_fetcher.py` |
| Longbridge | 港股、美股 | 港股、美股 | 否 | 否 | 是 | `data_provider/longbridge_fetcher.py` |
| Finnhub | 美股 | 美股 | 否 | 否 | 是 | `data_provider/finnhub_fetcher.py` |
| Alpha Vantage | 美股 | 美股 | 否 | 否 | 是 | `data_provider/alphavantage_fetcher.py` |

新闻能力由 `SearchService` 管理，不属于上述行情 Fetcher。实际搜索供应商取决于配置，可包括 Tavily、博查、SerpAPI、SearXNG 等路径，必须以 `src/search_service.py` 和当前配置为准。

## 阅读注意

- “日线市场”来自管理器的已知支持表，不代表每个供应商在任意时刻都可用。
- “实时行情”表示项目中存在接入路径，不表示全部字段、交易时段和新鲜度完全一致。
- YFinance 的 A 股日线能力来自代码支持表；实际可用性仍受 Yahoo 符号、网络和数据覆盖影响。
- 本表目前手工维护。后续应把能力声明结构化，再由脚本生成并在 CI 中检查漂移。
